# components 目录说明

- StartButtion.jsx：顶部欢迎开关组件。展示用户 pid 问候语，提供开关（Switch）跳转到首页/关闭状态。
- ChromeHeader.jsx：“已定义的规则”的页头。通用页头组件，展示标题、左侧返回图标，适用于扩展页面顶部栏。
- ChangeProfile.jsx：用户偏好/规则弹窗编辑器，支持添加、修改、删除、搜索等操作；同步规则到后端接口与浏览器存储。
- Chatbot/Chatbot.jsx：聊天主界面，管理会话列表、消息发送/接收、启动新会话、展示机器人和用户消息，并在需要时弹出偏好修改。
- Chatbot/ChatHeader.jsx：“规则配置助手”、“查看工具操作”的页头。聊天页头，包含返回和“更多”入口。
- Chatbot/SessionList.jsx：聊天会话列表卡片，展示历史对话摘要、创建新会话入口。
- Chatbot/Chatbot.css：聊天界面样式，控制消息气泡、布局等。
